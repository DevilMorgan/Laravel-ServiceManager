## Installation

Command to run.

Install dependencies
```
$ composer install
```
Generate .env file
```
$ cp .env.example .env
```
Generate a Key
```
$ php artisan key:generate
```
Setup link to storage folder
```
$ php artisan storage:link
```
Run migration and seed
```
$ php artisan migrate:fresh --seed
```
